Dissertation

Eczema Scratch Detection via Wearable IMU Sensors

Investigated automatic detection of scratching behaviour using wearable IMU sensor data (accelerometer, gyroscope, magnetometer at 20 Hz). Compared traditional ML against Transformer-based deep learning architectures for binary sequence classification.

  • Custom wearable sensor dataset; sliding window preprocessing (size=40, step=20) with majority-label voting
  • Built Standard Transformer (2 layers, 4 heads) and Conformer (Conv1D + TransformerEncoder) in PyTorch
  • Zero false positives on flat/non-movement scenarios; detected 20 scratch events on validation data
  • Sensitivity analysis across window/step sizes via heatmap grid search

ML Pipeline

Customer Churn Prediction in Telecom

End-to-end ML pipeline to predict customer churn for a telecom provider using 7,043 customer records. Integrated business intelligence to derive actionable retention strategies.

  • Applied SMOTE for class imbalance; engineered features from demographics, usage, and contract data
  • Compared Logistic Regression, Random Forest (n=100), Decision Tree via Accuracy, Confusion Matrix, ROC-AUC
  • K-Means clustering (k=3): Loyal Long-Term, Price-Sensitive, and High-Risk Short-Term customer segments
  • Key churn drivers: tenure under 20 months, month-to-month contracts, monthly charges over $60

NLP / Deep Learning

Political Bias Detection in News Articles

Multi-class NLP system classifying political bias (Left, Lean Left, Centre, Lean Right, Right) in 3,458 news articles using deep learning.

  • Text preprocessing: URL removal, stopword analysis; RandomOverSampler for class balance
  • Features via TF-IDF (5,000 features) and Keras Tokenizer with sequence padding (maxlen=300)
  • LSTM: Embedding(5000 to 128), LSTM(100, dropout=0.3), Dense(128, ReLU), Softmax output
  • Achieved 88% test accuracy; strongest on clearly partisan categories (Left, Right)

Data Mining

Loan Approval Prediction

ML pipeline to predict loan approval or rejection using 45,000 applications covering demographics, financial profiles, and loan characteristics.

  • Log transformation on 7 skewed features; RobustScaler and SMOTE for class imbalance
  • Compared Logistic Regression, Random Forest, KNN (k=3) via 10-fold cross-validation and ROC-AUC
  • Random Forest achieved 91.8% accuracy; top predictors: loan_percent_income, loan_int_rate, prior defaults
  • Correlation-based feature selection: kept features with r above 0.1 with target variable
